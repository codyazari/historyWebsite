The Browser Wars

Before The Browser Wars

  • The World Wide Web was an incredible invention created in 1989 by Tim Berners-Lee. However, the web was extremely hard to navigate if you were not a tech geek.
  • This changed when a Illinois undergrad student, Marc Andreessen, came up with the idea that everyone should be able to use the internet. The result of this idea was the world's first graphical web Browser, Mosaic.
  • In 1993, Mosaic was posted online as a free gift to humanity.
  • The web browsers used today are descendants of Mosaic.

Who was involved?

  • The Browser war was a battle between NetScape and Microsoft for the market dominance of Internet Web Browsers.

NetScape

  • NetScape was created by Marc Andreessen, Jim Clark, Aleks Totic, and Rob McCool.
  • After leaving Silicon Graphics, Jim Clark was looking for the next best thing to invest in. That next best thing was Mosaic. When Clark was introduced to Mosaic, he reached out to Marc Andreessen about creating a software company.

Microsoft

  • In 1975, Bill Gates Dropped out of Harvard to build microsoft with Paul Allen
  • Gates' goal was for all PCs to use Microsoft Software.
  • This goal almost became a reality with 90% of PCs running Microsoft Software.
